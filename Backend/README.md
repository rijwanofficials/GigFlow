# AI Docs Backend

Backend service for an AI-powered document understanding platform.

## Tech Stack
- Node.js + TypeScript
- Express.js
- MongoDB (documents, logs)
- PostgreSQL (users, auth)
- Prisma ORM

## Features (In Progress)
- User authentication
- Document upload
- AI-powered Q&A (RAG)
- Legal document analysis

## Setup
```bash
npm install
npm run dev
